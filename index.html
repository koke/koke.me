This is a website

Nothing fancy yet, just very basic HTML. Go on, use "View Source". I've been stuck for weeks trying to reboot my personal website. There are so many options these days, and so much complexity available.

The reality is that I want to start writing again, but I keep getting stuck picking the perfect tools. It never seems good enough, and I keep making excuses for not writing. So this is my website now, made with love from scratch.

You see, I got started building websites over 20 years ago, but when the iPhone came out, it caught my attention and I became an iOS developer for the following 15 years. It was fun, and I loved working with Swift, but now I'm back to building on the web.

Modern web development feels so complex these days that I really want to get back to basics, and get as close to HTML and CSS as possible (with a sprinkle of JS here and there). I start with no "backend", no static site generator, or even build tools. Just a plain old index.html file.
